About Engines, the Blogament README, and a quick Demo.

Duplication is the root of all evil! Engines can reduce errors and speed up development.

Gems = reusuable code

Engines = reusable code with a user interface (engine.rb = application.rb)

Reusuable classes are put in the engine's namespace, keeping the 'business domain' isolated.
