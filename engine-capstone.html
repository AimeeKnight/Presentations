Blogament


A Rails Engine designed to provide its host with blogging and commenting features.

Aimee Knight
June 27th, 2014

Final Capstone Goals:

  • Expand on my knowledge of Rails by creating my own gem, which is itself a Rails app. Learn how the magic works...
  • Explore ways of making applications more modular.
  • Dig deeper into TDD by relying heavily on RSpec Mocking, and the Capybara Webkit driver for testing JavaScript.

Technolody & Techniques, the Blogament README, and a quick Demo.

  • Searching with acts-as-taggable-on & Elasticsearch
  • Background jobs with SideKiq, RedisToGo and Unicorn
  • Duplication is the root of all evil!
    Engines can reduce errors and speed up development. Reusable classes are put in the engine's namespace and can then be isolated from the host application.
  • Gems = reusable code
  • Engines = reusable code with a user interface
    (engine.rb = application.rb)
